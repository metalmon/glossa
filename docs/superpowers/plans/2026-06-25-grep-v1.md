> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `grep` tool ŌĆö ripgrep-style literal/regex search over the text glossa already extracted (the stored chunk `body`), exact and File-First ŌĆö alongside the existing BM25 `search`.

**Architecture:** A new pure-Rust grep engine (`src/grep.rs`) compiles the pattern with the `regex` crate, selects candidate chunks (v1: full-scan all stored chunk bodies; an optional BM25 literal-prefilter narrows them), confirms with the real regex line-by-line, and emits `path:#n: line` where `#n` is the chunk's `ord` (the read key shipped in Plan A). Exposed via the MCP `grep` tool, the CLI `kb grep`, and mirrored in the kb-eval harness.

**Tech Stack:** Rust, tantivy 0.26 (stored chunk bodies), `regex` + `regex-syntax` (pure-Rust, the engine ripgrep uses).

**Scope:** Plan B of the spec `docs/superpowers/specs/2026-06-25-tool-contract-redesign-design.md` ┬¦3.3, per research `docs/superpowers/plans/glossa-grep-trigram-research.md` ┬¦2.A. The **trigram accelerator is Step 2 ŌĆö OUT OF SCOPE.** Context flags `-A/-B/-C` are also out of scope (they need cross-chunk File-First file re-reads) ŌĆö a documented follow-up.

## Global Constraints

- **C-free:** `cargo tree -p glossa -i cc` must stay empty. `regex`, `regex-syntax`, `aho-corasick`, `memchr` are all pure-Rust.
- **File-First:** grep searches the extracted text stored in the index (a disposable accelerator); the index can be rebuilt from files. v1 needs no file re-reads.
- **Reuse the `ord` contract from Plan A:** grep output addresses chunks as `#n` (the chunk `ord` ŌĆö pageŌä¢ for PDFs, section/sheet index otherwise), so `grep` ŌåÆ `read(path, n)` works exactly like `search` ŌåÆ `read`.
- **Uniform across formats:** grep runs over every chunk's extracted `body` (pdf/doc/docx/xlsx/md alike).

```rust
    /// Visit every stored chunk: `f(path, ord, file_type, body)`. Used by grep's full scan.
    pub fn iter_chunks(&self, mut f: impl FnMut(&str, u64, &str, &str)) -> anyhow::Result<()> {
        use tantivy::collector::DocSetCollector;
        use tantivy::query::AllQuery;
        let searcher = self.reader.searcher();
        let docs = searcher.search(&AllQuery, &DocSetCollector)?;
        for addr in docs {
            let d: TantivyDocument = searcher.doc(addr)?;
            let s = |fld| d.get_first(fld).and_then(|v| v.as_str()).unwrap_or("");
            let ord = d.get_first(self.fields.ord).and_then(|v| v.as_u64()).unwrap_or(0);
            f(s(self.fields.path), ord, s(self.fields.file_type), s(self.fields.body));
        }
        Ok(())
    }
```

```rust
/// Translate a shell glob (`*`, `?`) into an anchored regex over the whole path.
fn glob_to_regex(glob: &str) -> Result<regex::Regex, regex::Error> {
    let mut re = String::from("(?s)^");
    for ch in glob.chars() {
        match ch {
            '*' => re.push_str(".*"),
            '?' => re.push('.'),
            c => re.push_str(&regex::escape(&c.to_string())),
        }
    }
    re.push('$');
    regex::Regex::new(&re)
}

/// Build the line matcher from the pattern + flags. `-F` escapes the whole pattern, `-w` wraps it
/// in word boundaries, `-i` (smart-case) folds when the pattern has no uppercase.
fn build_matcher(pattern: &str, opts: &GrepOpts) -> anyhow::Result<regex::Regex> {
    let mut body = if opts.fixed { regex::escape(pattern) } else { pattern.to_string() };
    if opts.word {
        body = format!(r"\b(?:{body})\b");
    }
    let smart_case = opts.ignore_case && !pattern.chars().any(|c| c.is_uppercase());
    let re = RegexBuilder::new(&body).case_insensitive(smart_case).build()?;
    Ok(re)
}

pub fn grep(idx: &DocIndex, pattern: &str, opts: &GrepOpts) -> anyhow::Result<Vec<GrepHit>> {
    let matcher = build_matcher(pattern, opts)?;
    let glob_re = match &opts.glob {
        Some(g) => Some(glob_to_regex(g)?),
        None => None,
    };
    let mut hits = Vec::new();
    idx.iter_chunks(|path, ord, file_type, body| {
        if let Some(ft) = &opts.file_type {
            if file_type != ft { return; }
        }
        if let Some(gr) = &glob_re {
            if !gr.is_match(path) { return; }
        }
        for line in body.lines() {
            if matcher.is_match(line) {
                hits.push(GrepHit { path: path.to_string(), ord, line: line.trim().to_string() });
            }
        }
    })?;
    Ok(hits)
}
```

```rust
    /// Visit only chunks whose `body` contains `literal` as a whole BM25 token (a cheap superset
    /// of a substring match when the literal aligns with token boundaries). Returns false without
    /// visiting anything when `literal` is empty (caller must full-scan instead).
    pub fn iter_chunks_matching_term(
        &self,
        literal: &str,
        mut f: impl FnMut(&str, u64, &str, &str),
    ) -> anyhow::Result<bool> {
        use tantivy::collector::DocSetCollector;
        use tantivy::query::QueryParser;
        if literal.trim().is_empty() {
            return Ok(false);
        }
        let searcher = self.reader.searcher();
        let parser = QueryParser::for_index(&self.index, vec![self.fields.body]);
        let query = match parser.parse_query(&format!("\"{}\"", literal.replace('"', " "))) {
            Ok(q) => q,
            Err(_) => return Ok(false),
        };
        let docs = searcher.search(&query, &DocSetCollector)?;
        for addr in docs {
            let d: TantivyDocument = searcher.doc(addr)?;
            let s = |fld| d.get_first(fld).and_then(|v| v.as_str()).unwrap_or("");
            let ord = d.get_first(self.fields.ord).and_then(|v| v.as_u64()).unwrap_or(0);
            f(s(self.fields.path), ord, s(self.fields.file_type), s(self.fields.body));
        }
        Ok(true)
    }
```

In `src/grep.rs`, extract the longest required literal from the pattern and try the prefilter, falling back to the full scan. Add this helper and use it in `grep`:

```rust
/// The longest exact literal required by every match of `pattern` (used to prefilter via the
/// word-token index). None when the regex has no useful required literal (-> full scan).
fn required_literal(pattern: &str, opts: &GrepOpts) -> Option<String> {
    if opts.ignore_case && pattern.chars().any(|c| c.is_uppercase()) {
        // mixed-case + case-insensitive: token-level prefilter would be unsound; skip it.
        return None;
    }
    let pat = if opts.fixed { regex::escape(pattern) } else { pattern.to_string() };
    let hir = regex_syntax::parse(&pat).ok()?;
    let mut ext = regex_syntax::hir::literal::Extractor::new();
    ext.kind(regex_syntax::hir::literal::ExtractKind::Prefix);
    let seq = ext.extract(&hir);
    let lits = seq.literals()?;
    // Require an exact, non-cut prefix set so the literal is genuinely required by all matches.
    if !seq.is_exact() || lits.is_empty() {
        return None;
    }
    // The literals are alternative required prefixes; pick the first that is long enough (Ōēź3
    // chars) to be a selective whole-token query. If none qualifies, return None -> full scan.
    lits.iter()
        .map(|l| String::from_utf8_lossy(l.as_bytes()).to_string())
        .find(|s| s.chars().count() >= 3)
}
```

```rust
pub fn grep(idx: &DocIndex, pattern: &str, opts: &GrepOpts) -> anyhow::Result<Vec<GrepHit>> {
    let matcher = build_matcher(pattern, opts)?;
    let glob_re = match &opts.glob { Some(g) => Some(glob_to_regex(g)?), None => None };
    let mut hits = Vec::new();
    let mut visit = |path: &str, ord: u64, file_type: &str, body: &str| {
        if let Some(ft) = &opts.file_type { if file_type != ft { return; } }
        if let Some(gr) = &glob_re { if !gr.is_match(path) { return; } }
        for line in body.lines() {
            if matcher.is_match(line) {
                hits.push(GrepHit { path: path.to_string(), ord, line: line.trim().to_string() });
            }
        }
    };
    let prefiltered = match required_literal(pattern, opts) {
        Some(lit) => idx.iter_chunks_matching_term(&lit, &mut visit)?,
        None => false,
    };
    if !prefiltered {
        idx.iter_chunks(&mut visit)?;
    }
    Ok(hits)
}
```

```rust
/// Run a ripgrep-style exact/regex search over the extracted text; one line per match `path:#n: line`.
pub fn run_grep(idx: &DocIndex, pattern: &str, opts: glossa::grep::GrepOpts, trace: &TraceLog) -> (String, Vec<String>) {
    match glossa::grep::grep(idx, pattern, &opts) {
        Ok(hits) => {
            let titles: Vec<String> = hits.iter().map(|h| format!("#{}", h.ord)).collect();
            trace.log("grep", json!({ "pattern": pattern }), json!({ "hits": hits.len() }));
            let body = if hits.is_empty() {
                "(no matches)".to_string()
            } else {
                hits.iter().map(|h| h.display_line()).collect::<Vec<_>>().join("\n")
            };
            (body, titles)
        }
        Err(e) => (format!("grep error: {e}"), Vec::new()),
    }
}
```
